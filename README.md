# ROSERIA Jewelry Website Guide

## How to Run
Simply open `index.html` in your web browser (Chrome, Edge, etc.). No installation needed!

## How to Manage Products
1.  **Add your photo**: Place your product image inside the `assets/images` folder.
2.  **Update the list**: Open `js/products.js` in a text editor (Notepad, VS Code).
3.  **Add a new entry**: Copy the format of an existing product and add it to the list.

## How to Change Website Images
- **Hero Background**: Replace `assets/images/hero-bg.jpg` with your own photo.
- **Hero Featured Image**: Replace `assets/images/hero-jewelry.jpg` with your own photo.
- **About Section Image**: This is currently a web link. You can change it in `index.html` (search for "About Section").

## How to Update Contact Info
- **Phone/Email**: Open `index.html` and `contact.html` and search for the phone number (`777 7777`) to change it.

## Features
- **Premium Design**: Black & Gold Luxury Theme with Split Layout Hero.
- **Mobile Responsive**: Works perfectly on phones.
- **Shopping Cart**: Saves items automatically.
- **Online Store Mode**: No physical address displayed.
